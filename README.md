# report-node
Endpoint for "report-react", Used Node.Js, Expres.Js, and MongoDB.


## Available Scripts

In the project directory, you can run:

### `npm install`
### `npm run dev`

Open [http://localhost:5000](http://localhost:5000) to view it in the browser.
